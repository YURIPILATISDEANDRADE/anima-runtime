Expressions

Expressions are the building blocks of your code. These can be the following:

Unary Expressions: Self-explanatory, only has one argument.

Binary Expressions: Wrapped within parens, comprised of a left-hand side and a right-hand side.

Ternary Expressions: Also wrapped within parens, has three arguments.

Variadic Expressions: Also wrapped within parens, has two or more arguments.

Function calls: Like ones seen previously. Arguments are passed within parens, and separated with commas.

Block expressions: Surrounded in curly brackets, contains zero or more expressions, each ending in a semicolon.

Declarations: Comes in many shapes and sizes, those being:

  • Variable declarations
  • Function declarations
  • Namespace declarations
  • Macro declarations
  • Type extension declarations
  • Trait declarations

Macro Expansions: Self-explanatory, will be explained further in the macros section.
