All types in Breve are checked at compile-time for type safety. Numeric types are automatically converted between each other on any operation.

To assign the default value for a given basic type, you can use the provided value in the "Default Value" column, or you can simply assign the type itself:

// Assigns an unsigned integer of value 0
var1 = 0;

// Also assigns an unsigned integer of value 0
var1 = uint;

The any Type

The any type is a special compiler type that indicates that the data can map to the value of any other type.

While it can receive any value, only any types can assign to other any types without casting.

Casting

Casting allows the programmer to convert a value of a given type to another. it is done with the as operator.

example: int = (2.08 as int);

Both the source and destination types are checked if they are compatible. any is automatically assumed to be castable.

Caution!

Be careful when casting from any to any type! Make sure you check at runtime that the destination type is the expected type!
